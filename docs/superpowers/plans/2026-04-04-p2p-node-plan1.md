# GOAMP P2P Node — Plan 1: Core Skeleton

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a runnable `goamp-node` binary that Tauri can spawn as a sidecar — HTTP REST + WebSocket API, SQLite storage, plugin system — no P2P yet (P2P is Plan 2).

**Architecture:** `goamp-node/` lives inside the GOAMP monorepo as a Go module (`github.com/goamp/sdk`). Tauri bundles the binary and spawns it on startup. The node prints `ready:PORT` to stdout when the API is up. All inter-process communication is HTTP + WebSocket on localhost.

**Tech Stack:** Go 1.22+, modernc.org/sqlite (no CGO), gorilla/websocket, BurntSushi/toml, google.golang.org/grpc, google.golang.org/protobuf, testify, Tauri v2 sidecar API.

### Task 5 — 👤 YOU: Identity (ed25519 key load/generate)

**File:** `identity/identity.go`, `identity/identity_test.go`

**Contract:**

```go
// LoadOrGenerate loads the keypair from path.
// If path doesn't exist, generates a new ed25519 keypair and saves it.
func LoadOrGenerate(path string) (crypto.PrivKey, error)

// PeerID derives the libp2p peer.ID from a private key.
func PeerID(priv crypto.PrivKey) (peer.ID, error)
```

Use `github.com/libp2p/go-libp2p/core/crypto`:
```go
// Generate:
priv, _, err := crypto.GenerateEd25519Key(rand.Reader)
// Marshal:
b, err := crypto.MarshalPrivateKey(priv)
// Unmarshal:
priv, err := crypto.UnmarshalPrivateKey(b)
// PeerID:
pid, err := peer.IDFromPrivateKey(priv)
```

**Tests to write:**
- Call twice with same path → same peerID both times
- Delete file, call again → new key generated (different peerID)
- Generated key file has 0600 permissions

### Task 6 — 👤 YOU: SQLite migrations + Store CRUD

**Files:** `store/migrations.go`, `store/sqlite.go`, `store/sqlite_test.go`

**Schema** (copy from spec, implement in `migrations.go` as a `const schema string`):
```sql
CREATE TABLE IF NOT EXISTS tracks (
    id TEXT PRIMARY KEY,
    musicbrainz_id TEXT,
    acoustid TEXT,
    artist TEXT NOT NULL,
    title TEXT NOT NULL,
    duration_secs INTEGER,
    genre TEXT,
    peer_count INTEGER DEFAULT 1,
    last_seen INTEGER,
    created_at INTEGER DEFAULT (unixepoch())
);
CREATE TABLE IF NOT EXISTS providers (
    track_id TEXT NOT NULL,
    peer_id TEXT NOT NULL,
    announced_at INTEGER,
    PRIMARY KEY (track_id, peer_id)
);
CREATE TABLE IF NOT EXISTS peers (
    peer_id TEXT PRIMARY KEY,
    addrs TEXT NOT NULL,
    node_version TEXT,
    protocols TEXT,
    last_seen INTEGER,
    reputation INTEGER DEFAULT 0
);
CREATE TABLE IF NOT EXISTS peer_profiles (
    profile_hash TEXT PRIMARY KEY,
    profile_data TEXT NOT NULL,
    received_at INTEGER DEFAULT (unixepoch())
);
CREATE TABLE IF NOT EXISTS recommendation_cache (
    track_id TEXT PRIMARY KEY,
    score REAL NOT NULL,
    source TEXT NOT NULL,
    cached_at INTEGER DEFAULT (unixepoch())
);
```

Use `modernc.org/sqlite` (pure Go, no CGO):
```go
import (
    "database/sql"
    _ "modernc.org/sqlite"
)
db, err := sql.Open("sqlite", filepath.Join(dataDir, "node.db"))
// For in-memory (tests):
db, err := sql.Open("sqlite", ":memory:")
```

**Methods to implement on `SQLiteStore`:**

## Phase 2 — HTTP API

### Task 7 — 👤 YOU: HTTP server + /health + WebSocket /events

**Files:** `api/server.go`, `api/health.go`, `api/websocket.go`, `api/server_test.go`

**`server.go`** — create `Server` struct, register all routes, `Start(addr string) error`:
```go
type Server struct {
    node    sdk.Node
    catalog sdk.Catalog
    profiles sdk.ProfileAggregator
    plugins *plugin.Loader
    hub     *WSHub  // WebSocket broadcast hub
}

func New(node sdk.Node, catalog sdk.Catalog, profiles sdk.ProfileAggregator, plugins *plugin.Loader) *Server

func (s *Server) Start(addr string) error  // starts http.ListenAndServe
```

Routes to register:
```
GET  /health
GET  /peers
GET  /catalog/search
POST /catalog/announce
POST /profiles/sync
GET  /recommendations
GET  /events              (WebSocket)
GET  /plugins
*    /plugins/{id}/
```

**`health.go`** — handler for `GET /health`:
```json
{"status":"ok","peer_count":0,"uptime_secs":42,"mode":"client","version":"0.1.0"}
```

**`websocket.go`** — `WSHub` with `Broadcast(event sdk.Event)` and `ServeWS(w, r)`. When Tauri connects to `/events`, it receives JSON events pushed from the node. The Node's `Emit(event)` method calls `hub.Broadcast(event)`.

**Tests:** `httptest.NewServer`, test `/health` returns 200 + valid JSON.

## Build

Once Plan 1 is complete, build the sidecar:

```bash
cd goamp-node
GOOS=linux GOARCH=amd64 go build -o ../src-tauri/binaries/goamp-node-x86_64-unknown-linux-gnu ./cmd/goamp-node
```

Then `pnpm tauri build` bundles it.

For development (no Tauri build):
```bash
cd goamp-node && go run ./cmd/goamp-node --mode=client --api-port=7472
curl localhost:7472/health
```
